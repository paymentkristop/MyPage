Discrepancy Reports In (Receipts)

1
Initial Action

On receipt of MOD Form 445 (DR), check timescales.
Timescale: 21 Days (UK), 42 Days (Overseas).
Enter into DR In Register immediately.

2
Investigation

1. Check Transaction Record (Issues).
2. Physical check of stock locations.
3. Check VITAL/Trackers (Note: VITAL node arrival is NOT proof of delivery for DR purposes).

3
Resolution

Liability Admitted: Adjust store account (Certificate Receipt/Issue). Write off value in Losses Register. Annotate F445 "LIABILITY ADMITTED".
Rejected: Annotate "TIME EXPIRED - CONSIGNEE LIABILITY" if outside window.

Planned Repair (PR) & ES Mat

1
Identification

PR items are Permanent (P Class). Look for Inventory Classification Codes (ICC):
B: Repairable, tracked by qty.
S: CLS Repairable.

2
The Repair Loop (DIINs)

All PR items due into depot generate a DIIN (Dues In ID Number).
Prefixes:
A = Demand Generated
Y = Unit Disposal Request
X = DT Disposal Request
PR = Direct to Repair (DTR)

3
MIR226 Management

Units MUST monitor the VERITAS MIR226 report.
1. Upload Issue Voucher (IV) details to clear the line.
2. Upload VITAL references.
3. If cleared receipt not received, follow Hastening process.

Disposals

1
Application

1. Condition item (E0).
2. Request Disposal Instructions via MJDI.
3. Wait for U053 (Disposal Instruction Response).

2
Responses

Backload: Issue to Depot (Creates DIIN).
On Unit Disposal (AF G8609): Strike off charge (GXD). Use local scrap metal contract or DRDT if specified.
